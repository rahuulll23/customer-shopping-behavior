# customer-shopping-behavior
Data Analytics Project

📊 Customer Behavior Analysis & Sales Dashboard
📌 Overview

This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI to uncover valuable business insights.
The complete workflow includes:

✔️ Data Loading
✔️ Data Cleaning
✔️ Exploratory Data Analysis (EDA)
✔️ SQL Query Analysis
✔️ Dashboard Development
✔️ Report & Presentation Creation

The final outcome is an interactive Power BI Dashboard that helps understand customer purchasing trends, revenue distribution, subscription behavior, and category-wise sales performance.

📂 Dataset

The dataset contains customer shopping and transaction information such as:

👤 Customer Demographics
💳 Subscription Status
🛍️ Product Categories
💰 Purchase Amounts
⭐ Review Ratings
🚚 Shipping Types
📈 Sales & Revenue Details

The dataset was analyzed to identify trends and support data-driven decision-making.
